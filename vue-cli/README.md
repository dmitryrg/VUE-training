# vue-training

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run local API server for development
```
npm run api
```

### Run your tests
```
npm run test
```

### Compiles and minifies for production (optional)
```
npm run build
```

### Lints and fixes files (optional)
```
npm run lint
```
